# 人才画像系统

## 项目概述

这是一个基于.NET 8和MySQL的人才画像系统，用于分析公司员工数据，构建个人画像和集体画像，帮助公司了解人才结构是否合理，哪类人才冗余，哪类人才稀缺等。

## 技术栈

- 后端：ASP.NET Core 8.0
- 数据库：MySQL 8.0
- ORM：Entity Framework Core
- 前端：Bootstrap、Chart.js

## 主要功能

1. **员工个人画像**：查看员工的详细信息，包括基本信息、标签、综合能力评分、能力雷达图等。
2. **高绩效人才池**：按照综合能力评分排序，展示公司高绩效人才。
3. **集体画像**：包括人才分布统计、人才结构分析、突出员工一览等。

## 系统架构

- **数据访问层**：使用Entity Framework Core连接MySQL数据库
- **业务逻辑层**：包含各种服务类，实现数据处理和分析
- **表示层**：MVC架构的控制器和视图

## 运行项目

1. 确保已安装MySQL 8.0并已创建员工数据库
2. 确保已安装.NET 8 SDK
3. 运行项目：
   ```
   dotnet run
   ```
   或双击`run.bat`批处理文件

## 数据库结构

系统使用的数据库包含以下主要表：
- employees：员工基本信息
- attendance_records：考勤记录
- position_changes：职位变动记录
